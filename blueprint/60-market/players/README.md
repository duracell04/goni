# Other Players Lane

Use this lane for important ecosystem actors that are not direct competitors or direct suppliers.

Examples:
- adjacent platforms
- standards groups
- infrastructure projects
- distribution channels

Focus:
- architecture and technical direction
- trust model and governance implications
- relevance to Goni's roadmap

## Goni interpretation

- UI, channel, and workflow systems are replaceable front doors.
- Model/runtime systems are replaceable backends.
- Tool protocols and sandboxes are integration surfaces.
- The Goni-specific value remains the governed kernel: policy, capability
  mediation, receipts, rollback, and autonomy corridors.
