# EVID-SEC-02 Memory Poisoning Lane

Goal: evaluate whether untrusted or adversarial content is incorrectly promoted
into durable memory.

## Core checks
- poisoning acceptance rate
- false rejection rate for benign updates
- time to quarantine/revert poisoned entries
- audit traceability from source to memory delta
