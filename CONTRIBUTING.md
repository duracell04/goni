# CONTRIBUTING

Quick start:
- See docs/quickstart.md for setup and demo steps.
- Run tests with: make test
- Run the smoke test with: make smoke

Spec changes:
- If you change docs/specs, add an RFC in docs/rfcs/ and update TRACEABILITY.md.
- If you change schemas, update software/50-data/51-schemas-mvp.md and
  software/50-data/53-schema-dsl-and-macros.md (see AGENTS.md).

Evidence:
- Claims of enforcement require tests or benchmarks.
