# CONTRIBUTING

Quick start:
- See blueprint/docs/quickstart.md for setup and demo steps.
- From blueprint/, run tests with: make test
- From blueprint/, run the smoke test with: make smoke

Spec changes:
- If you change blueprint/docs/specs, add an RFC in blueprint/docs/rfcs/ and update TRACEABILITY.md.
- If you change schemas, update blueprint/software/50-data/51-schemas-mvp.md and
  blueprint/software/50-data/53-schema-dsl-and-macros.md (see AGENTS.md).

Evidence:
- Claims of enforcement require tests or benchmarks.
